# swapi.dev

A few tests for [The Star Wars API](https://swapi.dev/documentation).


## Technologies
- Kotlin
- Maven
- JUnit
- Gson
- REST Assured


## How to run tests
You should have JVM on your machine
- Download project
- Using terminal/CMD go to project folder
- Execute:

Unix-like:
```
./mvnw test
```
Windows:
```
mvnw test
```
